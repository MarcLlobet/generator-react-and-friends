# generator-react-and-friends
A boilerplate generator for react projects.

### Installation
If you haven't yet, create the folder of your new project, place yourself there:
```
mkdir my-new-project
cd my-new-project
```

Run the generator: 
```
yo react-and-friends
```

## Stack
 * Webpack
   - Dev server
   - Optimization
   - HTML template
 * Javascript
   - Babel
   - Typescript
 * React
   - JSX
 * CSS
   - CSS modules
   - SCSS
 * Linters / Formatters
   - Eslint
   - Prettier
   - Stylelint
 * Testing
   - Jest
   - Enzyme
   - React test renderer
